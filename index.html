Para fazer o seu site funcionar 100% grátis e sem cartão de crédito, precisa configurar a sua conta no Supabase.

  1. Vá a supabase.com e crie um projeto novo.
  2. No menu esquerdo, vá a Project Settings > API e copie a sua URL e anon key.
  3. Cole essas chaves no código-fonte deste ficheiro (linha ~520).
  4. Vá a SQL Editor, crie uma "New Query", cole o código abaixo e clique em Run:
create table cpr_users (id text primary key, data jsonb); create table cpr_matches (id text primary key, data jsonb); create table cpr_queue (id text primary key, data jsonb); create table cpr_chat (id text primary key, data jsonb); alter publication supabase_realtime add table cpr_users, cpr_matches, cpr_queue, cpr_chat;
